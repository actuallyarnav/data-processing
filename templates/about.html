About

Cloud-based data processing and analysis platform

Project

Data Proc is a web-based data analysis platform designed to simplify EDA for structured datasets. It allows users to upload datasets, configure analysis parameters, and generate statistical summaries and insights.

Technology stack
  • Flask — backend web framework
  • Pandas & NumPy — data processing and analysis
  • Bootstrap — frontend layout and styling
  • Docker — containerized deployment
  • AWS — cloud hosting infrastructure
