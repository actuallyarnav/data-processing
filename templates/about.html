About the Project

A cloud-based data processing and analytics platform

Data Proc is a cloud-hosted data analytics platform designed to simplify exploratory data analysis for structured datasets. Users can upload CSV files, configure analysis options, and generate statistical insights and visual summaries through a user-friendly web interface.

The platform emphasizes modular design, scalability, and usability, combining modern web technologies with Python-based data processing tools.

Technologies Used
  • • Flask (Backend Framework)
  • • Bootstrap (Frontend UI)
  • • Pandas & NumPy (Data Processing)
  • • Docker (Containerization)
Project Objectives
  • • Enable intuitive dataset upload and analysis
  • • Provide schema-aware statistical insights
  • • Maintain modular and scalable architecture
  • • Support cloud-based deployment
